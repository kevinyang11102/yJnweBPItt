# 基于Python的MNIST手写数字识别系统 python1142

## 项目概述

本项目是一款基于Python语言开发的MNIST手写数字识别系统，采用PyTorch作为主要的技术栈。系统支持数据的加载和鼠标手写绘图功能，具备图形用户界面（GUI）显示能力，用于展示识别过程和结果。

## 技术栈

- **后端**: Python 3.x
- **深度学习框架**: PyTorch
- **图形界面**: GUI支持

## 功能模块

- **数据加载**: 支持加载标准的MNIST数据集
- **绘图输入**: 允许用户通过鼠标绘制数字进行识别
- **模型训练**: 内置模型训练模块，可进行模型训练和评估
- **结果展示**: 可展示识别结果以及以下可视化内容
  - 损失曲线图
  - 准确率曲线图
  - 混淆矩阵图

## 系统角色

系统主要面向以下用户角色：
- 数据科学家/开发者：可使用系统进行模型训练、评估和调试
- 终端用户：通过GUI界面使用手写数字识别功能

## 运行环境

- 操作系统: 兼容Linux、Windows、macOS
- 硬件要求: 无特殊要求，常规配置即可
- 软件依赖: Python 3.x, PyTorch

## 部署步骤

1. 确保Python环境和PyTorch已正确安装
2. 克隆或下载项目源码
3. 安装依赖项（如需）
4. 运行主程序

## 目录结构

```
project/
│
├── data/               # 存放MNIST数据集
├── src/                # 源代码目录
│   ├── model.py        # 定义模型结构
│   ├── train.py        # 训练模型
│   ├── utils.py        # 工具函数
│   └── main.py         # 主程序
│
├── reports/            # 存放实验报告

```
